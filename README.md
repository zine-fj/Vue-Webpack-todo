#### 前言
此源码基于 [慕课网](https://www.imooc.com/) Jokcy老师的 [Vue+Webpack打造todo应用](https://www.imooc.com/learn/935) 视频教程编写。 

#### 使用步骤

##### 安装依赖
- 安装NPM依赖：`npm install`


##### 开发环境
- 启动项目：`npm run dev`
- 浏览器预览：__localhost:8888__


##### 生产环境
- 启动项目打包：`npm run build`
